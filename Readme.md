## Python 설치

- https://www.python.org/downloads/ 홈페이지 접속 후 다운로드 탭을 선택하고, 현재 버전(3.10.2) 다운로드를 합니다.
- 다운로드 완료 후 설치를 시작합니다. **Customize installation** 클릭하고, **install location** 란에 `c:\Python310` 입력 후 설치 버튼을 클릭합니다.
